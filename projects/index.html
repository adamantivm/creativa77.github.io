Projects

Monitoring price changes around the world

Pricespot is the mobile application used to gather consumer prices and product data for premise, a company devoted to track macroeconomic and human trends in real time.

We improved the Android mobile application design with the use of Android Services for data upload, enhanced UI interaction and integrated it with Google Places API for better location support.

Bringing personal robots to the service industry

This project was a collaboration between Ekumen and Willow Garage to take their personal robots to the next level and turn them into products for customers such as hospitals or restaurants.

We contributed with our expertise on system integration for bringing some of the existing modules and projects into the services market. We defined the system architecture and adapted the development infrastructure from an open source research project to a private product development business.

At the UX level we created intuitive web interfaces for complex robotics problems using Ember.js, Web Sockets and the Robot Web Tools bundle. We also tackle the problem of non-expert users controlling robots under continuous operation and end-user robot programming.

At the backend level we worked throughout the whole ROS stack using C++ and Python to implement multiple robot networks (each one with its own master node) and extended the python client API. We also performed Kinematics analysis on different client environments using MoveIt!.

Real-time transcription by non-expert captionist crowdsourcing

Scribe collects audio from a mobile device and schedules workers to perform captioning on small audio segments.

The distributed captions are then merged into a single text, returning a consolidated transcript to the mobile user.

We worked with Adobe Air on the mobile client to stream audio to an Adobe Media Server and with Javascript and AJAX in the front-end worker application. Back-end development included creating diagnostic and debugging tools in a LAMP Stack as well as integrating with the Java process in charge of merging the distributed transcriptions.

Live online video streaming

We are prototyping new content protection solutions for live video delivered over the Internet.

For this project we used Adobe Media ServerAdobe Media Server instances running in the Amazon CloudAmazon Cloud and stream content via HDS (HTTP Dynamic Streaming). The backend processing is implemented in Python and uses MongoDB for persistence.

Inventing the next-generation user interfaces for autonomous robots

Savioke mission is to develop affordable, usable robots for the services industry.

At Ekumen we are helping Savioke to develop top-quality user experiences for robot control, including autonomous navigation and teleoperation. Our approach is based on web user interfaces using Meteormeteor, Javascript and Rosbridge, a JSON transport layer included in the Robot Web Tools bundle.

Porting ROS to Android devices

The aim of this project is to develop an autonomous robot system using ROS that executes entirely on an Android device.

We contributed to ROS Java by implementing a base controller from scratch, extend it to use the multiple radios instead of just WiFi and added the ability to create externally reachable masters in an Android device. We are also diving into ROS internals to isolate the minimum required modules for a robot to navigate autonomously. Depending on the situation we are re-implementing core functionality in Java or compiling existing sources with Android NDK.

Implementing language bindings for airbrite.io

Airbrite.io exposes a full e-commerce sollution to developers by means of a REST API, including product catalog, orders and online payments.

We created API bindings for Backbone.js and Python and we are currently working on Node.js, Ruby and PHP versions.
